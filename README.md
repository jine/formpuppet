# formpuppet

A simple script in TypeScript, using Puppeteer to fill in a form and submit it on a fixed schedule.

## Features

- Automated form filling using Puppeteer
- Scheduled execution
- Written in TypeScript

## Getting Started

1. Install dependencies: `npm install`
2. Configure your form details and schedule in `main.ts`
3. Run the script: `npm run start` or `npx ts-node main.ts`

## Usage

Edit `main.ts` to specify the form URL, fields to fill, and the schedule for submission.
